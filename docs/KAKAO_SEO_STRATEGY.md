### PHASE 5: 연관 검색어 전략

#### Google 연관 검색어 노출 방법

**원리:**
```
사용자가 "카카오톡" 검색 시
→ Google이 연관 검색어 추천
→ "카카오톡 대화 분석" 노출
→ 클릭 → 속마음 스캐너
```

**실행 방법:**

1. **키워드 동시 사용**
   - "카카오톡" + "대화 분석"을 문서 내 50회 이상 사용
   - 자연스럽게 배치

2. **Click-Through Rate (CTR) 향상**
   - 매력적인 Title: "카카오톡 대화 분석 무료 사이트 | AI가 10초만에 분석"
   - 매력적인 Description: "썸인지 어장인지 궁금하다면? 카톡 대화만 업로드하세요!"

3. **체류 시간 증가**
   - 블로그 콘텐츠로 3분 이상 체류 유도
   - 동영상 임베드 (유튜브 사용법)

4. **이탈률 감소**
   - 관련 포스트 추천
   - "다음 읽을 글" 섹션
