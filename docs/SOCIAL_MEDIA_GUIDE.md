## 📊 플랫폼별 전략

### 1. Instagram (@mindscanner)

#### 계정 설정
```
📌 아이디: @mindscanner
📌 이름: 속마음 스캐너 | 카톡 대화 분석
📌 바이오:
    카톡 대화로 알아보는 우리 사이 💘
    ✨ AI가 10초만에 호감도 분석
    🔒 100% 익명 보장
    👇 무료 분석 시작
    🔗 [링크]

📌 프로필 사진: 속마음 스캐너 로고
📌 하이라이트:
    - 사용법
    - 분석 결과 예시
    - 후기
    - 연애 팁
```

#### 콘텐츠 전략 (주 3~5회 포스팅)

**포스트 유형:**

1. **분석 결과 예시** (가장 인기!)
   ```
   제목: "호감도 87점! 썸 타는 중 💕"
   이미지: 분석 결과 캡처 (개인정보 가림)
   캡션:
   ---
   "답장 속도 3분, 이모티콘 사용 多
   질문 빈도 높음... 이건 확실히 관심 있는 거죠? 😊

   당신의 카톡도 분석해보세요!
   프로필 링크에서 무료로 시작 →

   #카톡분석 #썸 #연애 #호감도테스트
   #MBTI #연애심리테스트 #커플 #데이트
   ```

2. **연애 심리 팁**
   ```
   제목: "답장 빠른 사람 vs 늦은 사람, 누가 더 좋아할까?"
   캐러셀 포스트 (슬라이드 10장)
   1장: 질문
   2~9장: AI 분석 결과 & 팁
   10장: CTA (무료 분석하기)
   ```

3. **Before & After**
   ```
   제목: "분석 전 vs 분석 후"
   Before: "이 사람 나한테 관심 있나...?"
   After: "호감도 92점! 확실한 호감 신호였네!"
   ```

4. **Meme & 릴레이터블 콘텐츠**
   ```
   제목: "카톡 답장 기다리는 나의 모습"
   GIF 또는 밈 이미지
   + 속마음 스캐너 소개
   ```

**해시태그 전략 (최대 30개):**
```
메인 해시태그:
#카톡분석 #카카오톡분석 #썸테스트 #연애심리테스트
#호감도테스트 #어장관리 #썸남 #썸녀

관련 해시태그:
#연애 #데이트 #커플 #밀당 #연애상담
#MBTI #20대 #대학생 #연애고민

트렌드 해시태그:
#데일리 #일상 #소통 #팔로우 #좋아요
```

#### 릴스 (Reels) 전략

**콘텐츠 아이디어 (15초 ~ 30초):**

1. **사용법 튜토리얼**
   ```
   0~5초: "카톡으로 누가 더 좋아하는지 알 수 있다?"
   5~10초: 화면 녹화 (파일 업로드)
   10~15초: 분석 결과 등장
   15초: "무료로 해보세요!"

   배경음악: 트렌디한 K-POP
   ```

2. **놀라운 분석 결과**
   ```
   0~3초: "헉...! 이 정도로 정확해?"
   3~10초: 분석 결과 클로즈업
   10~15초: "당신도 해보세요!"
   ```

3. **연애 팁 시리즈**
   ```
   주제: "호감도 높이는 카톡 답장법 5가지"
   포맷: 빠른 텍스트 전환 + 아이콘
   ```

```
    👇👇👇
```

#### 콘텐츠 전략

**TikTok에서 효과적인 포맷:**

1. **POV (Point of View)**
   ```
   제목: "POV: 속마음 스캐너로 분석한 결과"
   시나리오:
   - 분석 전: 불안한 표정
   - 분석 중: 로딩 화면
   - 분석 후: 놀라거나 기쁜 표정
   - 마지막: "링크 in bio"
   ```

2. **Storytime**
   ```
   "친구 카톡 분석해줬다가 충격받은 썰"
   텍스트 오버레이 + 리액션
   ```

3. **Tutorial**
   ```
   "카톡 대화 분석하는 법 (무료)"
   화면 녹화 + 보이스오버
   ```

**트렌드 활용:**
- 인기 사운드 사용
- 챌린지 참여
- 듀엣/스티치 활용

**해시태그:**
```
#카톡분석 #연애 #썸 #MBTI #연애테스트
#fyp #foryou #viral #korea #korean
```

#### 채널 설정
```

```

#### 콘텐츠 전략

**Shorts 아이디어 (60초 이하):**

1. **사용법 가이드**
   ```
   제목: "카톡 대화 내보내는 법 (초간단)"
   화면 녹화 + 자막 + BGM
   ```

2. **분석 결과 리뷰**
   ```
   제목: "호감도 95점 받은 카톡 대화 특징 TOP 5"
   텍스트 애니메이션 + 아이콘
   ```

3. **연애 심리 시리즈**
   ```
   시리즈명: "AI가 알려주는 연애 법칙"
   에피소드:
   - 1편: 답장 속도와 호감도의 상관관계
   - 2편: 이모티콘 사용량으로 보는 진심도
   - 3편: ㅋㅋㅋ 개수로 알아보는 감정
   ```

**SEO 최적화:**
- 제목: "카톡 대화 분석 | 호감도 테스트 | 무료"
- 설명: 블로그 링크 + 키워드 포함
- 태그: #카톡분석 #연애테스트 #MBTI

## 🎯 플랫폼별 우선순위

### 1순위: Instagram
- **이유:** 타겟층(20~30대) 많음, 시각적 콘텐츠 강함
- **목표:** 3개월 내 팔로워 1,000명
- **노력:** 주 5회 포스팅

### 2순위: TikTok
- **이유:** 바이럴 가능성 높음, 젊은 층
- **목표:** 첫 바이럴 영상 만들기 (조회수 10만+)
- **노력:** 주 3~4회 포스팅

### 3순위: 네이버 블로그
- **이유:** 한국 SEO에 강함, 백링크
- **목표:** 월 포스팅 8~10개
- **노력:** 주 2~3회 포스팅

### 4순위: YouTube Shorts
- **이유:** 장기적 성장, SEO
- **목표:** 구독자 500명
- **노력:** 주 2~3회 포스팅

### 5순위: Twitter/X
- **이유:** 실시간 소통, 트렌드
- **목표:** 팔로워 500명
- **노력:** 일 1~2회 트윗
